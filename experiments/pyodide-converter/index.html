Pyodide IFC → GLB

Loads Pyodide + the official ifcopenshell WASM wheel + trimesh in a Web Worker, then converts an IFC file to GLB entirely in the browser. No server-side conversion involved.

What this is testing
  • Whether ifcopenshell.geom tessellation works in Pyodide WASM (no pythonocc-core).
  • Whether trimesh can assemble the iterator output into a valid GLB.
  • Cold-start cost (Pyodide + ifcopenshell wheel size, first-load time).
  • Conversion speed on a real IFC vs the server pipeline.
